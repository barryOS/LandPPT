# 门窗行业业务调研与数字化解决方案报告

## 1. 行业背景与现状

门窗行业（特别是定制门窗）属于典型的**大行业、小企业**市场，具有“高客单价、低频消费、重服务、非标定制”的特点。

### 核心特征
*   **非标定制化**：几乎每家每户的窗户尺寸都不同，无法像家电一样标准化生产。
*   **服务链条长**：测量 -> 设计 -> 生产 -> 物流 -> 安装 -> 售后，任一环节出错都会导致严重客诉。
*   **价格不透明**：型材壁厚、玻璃品牌、五金配件、辅材（胶、发泡剂）等参数复杂，消费者难以比价。
*   **决策周期长**：消费者通常需要多次对比、到店体验才能下定。

## 2. 业务痛点分析

### C端（消费者）痛点
1.  **“所见非所得”**：展厅看到的样品与安装到自家的效果无法想象，缺乏直观感受。
2.  **价格陷阱**：低价吸引，后期通过增项（如转角立柱、辅材费）加价。
3.  **进度盲区**：付了定金后不知道工厂生产到哪一步了，安装时间一拖再拖。
4.  **售后无门**：安装漏水、五金损坏，找不到当初的经销商或门店倒闭。

### B端（经销商/门店）痛点
1.  **获客难**：线下自然进店流量枯竭，过度依赖老客户转介绍。
2.  **转化率低**：无法快速出图，口头描述方案缺乏说服力。
3.  **错单漏单**：手写测量数据字迹潦草，传给工厂时出错，导致产品报废。
4.  **管理混乱**：订单进度靠电话催工厂，安装工人调度靠微信群，效率低下。

## 3. 数字化解决方案 (App + Web)

为了解决上述痛点，建议构建一套 **“营销+工具+管理”** 的数字化系统。

### 3.1 角色划分
*   **C端用户 (业主)**：主要用于选品、看案例、查进度、售后报修。
*   **B端用户 (导购/经销商)**：主要用于CRM管理、测量录入、快速报价、下单。
*   **安装/服务人员**：查看安装任务、上传完工照片、处理售后。
*   **Web官网**：品牌形象展示、招商加盟入口、PC端管理后台。

## 4. 竞品参考
*   **皇派门窗、新豪轩**：强调品牌展厅、3D云设计、全屋定制概念。
*   **酷家乐/三维家**：提供前端设计工具，解决“所见即所得”问题。
*   **贝壳/土巴兔**：提供装修日记和口碑评价体系。

## 5. 建议开发路线
1.  **第一阶段 (MVP)**：Web官网展示 + 小程序(C端看款 + B端简单的CRM/报价)。
2.  **第二阶段**：引入简易绘图/测量工具，实现订单状态全链路打通。
3.  **第三阶段**：AR换窗体验，AI智能设计方案。
