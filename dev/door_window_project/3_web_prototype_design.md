# Web 端原型设计方案 (Official Website & Admin Portal)

## 1. 平台构成
1.  **前台官网 (Public Site)**：面向终端消费者和加盟商。
2.  **管理后台 (Admin Portal)**：面向总部运营、财务、工厂。

## 2. 前台官网设计 (Public Site)

### 2.1 站点地图 (Sitemap)
- **首页 (Home)**：Banner轮播、核心优势、爆款产品推荐、客户见证、预约入口。
- **产品中心 (Products)**：
    - 窗 (Windows)：平开窗、推拉窗、内倒窗。
    - 门 (Doors)：推拉门、平开门、折叠门。
    - 阳光房 (Sunrooms)。
- **案例展厅 (Inspiration)**：实景图库，支持按风格（现代、中式）、户型筛选。
- **品牌故事 (About Us)**：工厂VR全景、资质荣誉、发展历程。
- **服务支持 (Service)**：质保查询、防伪查询、保养知识。
- **招商加盟 (Franchise)**：加盟政策、利润分析、申请表单。

## 3. 管理后台设计 (Admin Portal)

### 3.1 功能模块
- **工作台 (Dashboard)**：待处理订单、今日销售额、生产进度预警。
- **CRM 系统**：全渠道线索管理（官网、抖音、线下）、公海池、分配机制。
- **OMS 订单系统**：
    - 订单审核（财务审、技术审）。
    - 拆单排产（对接工厂MES）。
    - 物流发货。
- **产品中心**：商品上架、SKU管理、价格策略配置。
- **财务管理**：经销商充值、货款扣除、发票管理。
